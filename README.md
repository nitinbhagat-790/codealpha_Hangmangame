# codealpha_Hangmangame

Hangman is a word guessing game in which a player tries to discover a hidden word by guessing one letter at a time. The computer selects a random word and displays blank spaces representing each letter. The player enters letter guesses, and if a guessed letter is present in the word, it is revealed in all its positions. If the guessed letter is not in the word, the player loses one chance. The game continues until the player correctly guesses the entire word or runs out of allowed attempts. This game is commonly used to improve vocabulary, spelling, logical thinking, and problem-solving skills. In programming, Hangman is a popular beginner project that demonstrates the use of loops, conditional statements, lists, strings, random word selection, and user input handling.
